Conceptos Básicos de CSS

¿Qué es CSS3?

En sus siglas es «Cascade Style Scheets» en español «hoja de estilos en cascada» versión 3, es conocido como lenguaje de diseño gráfico o estilos css, es utilizado para dar el formato de presentación visual a las páginas web. CSS 3 sirve generalmente para tranformar páginas web con diseños espectaculares con: colores, formato de texto, imágenes, animaciones, transiciones, diseños responsivos, etc.

¿Qué es HTML5?

Actualmente en la version 5 y cuyas siglas se lee «Hyper Text Markup Languaje» o en español Lenguaje de Marcas de Hipertexto. Es el lenguaje estandar de etiquetas utilizado para estructurar y organizar la información contenida en páginas web.

HTML5 y CSS3 están directamente relacionados a la estructúra y aspecto visual de las paginas web de internet, con los cuales interactuamos a travéz de los navegadores como: Chrome, Mozile Firefox, Opera, etc.

ATRIBUTO SIGNIFICADO
Colspan Combina la celda actual con el número de celdas a la derecha que se indique. Por ejemplo colspan=”3” une esta celda con las dos que tiene a su derecha, formando una combinación de tres celdas en horizontal.
rowspan Combina la celda actual con el número de celdas hacia abajo que se indique. Por ejemplo rowspan=”3” une esta celda con las dos que tiene hacia abajo, formando una combinación de tres celdas en vertical.
